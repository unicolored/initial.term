<initial-sh>

A console for webmasters worldwide

Console mode

Drop the <initial-sh> tag into your webpage with the script add the script to your page. Invoke the method show() of the initial-sh element when trigger an event, like a click on a button. By default, you can also press the Help key on your keyboard (= Insert on macos). To close the console: press Escape, or type exit.


<initial-sh id="console1" sounds></initial-sh>

<body>
...
<button onclick="document.getElementById('console1').show()">Open the console</button>

<initial-sh sounds></initial-sh>

<script type="module" src="https://cdn.jsdelivr.net/npm/initial-sh/initial-sh.bundled.js"></script>
</body>

Static mode

Another possibility is to display <initial-sh> into a custom container with a minimum of styling. You will pass static as an attribute. In this case, the console will be automatically opened.

<div class="my-custom-container" style="height: 337px; background: darkblue">
  <initial-sh id="console2" static></initial-sh>
</div>
